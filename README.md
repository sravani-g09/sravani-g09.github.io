# Sravani Guntupalli — Portfolio Site

A multi-page portfolio website. Static HTML, no build step, free to host.

## Files
- `index.html` — landing page (hero, about, skills, experience, education, case studies, contact)
- `case-shopify.html` — Case 01: WooCommerce → Shopify migration (flagship deep-dive)
- `case-operations.html` — Case 02: operations consolidation
- `case-reliability.html` — Case 03: enterprise reliability & cost

Each page is self-contained (styles are inline), so you can open any file directly in a browser to preview.

## Put it live on GitHub Pages (free, ~5 minutes)
1. Create a free account at github.com.
2. Click **New repository**. Name it `sravani-g09.github.io` (use your actual GitHub username in place of `sravani-g09`). Set it to **Public**. Create.
3. On the repo page, click **Add file → Upload files**. Drag in all four `.html` files (and this README if you like). Commit.
4. Go to **Settings → Pages**. Under "Branch," pick `main` and `/ (root)`, then **Save**.
5. Wait 1–2 minutes. Your site is live at `https://YOUR-USERNAME.github.io/`.

That URL goes in your LinkedIn "Featured" section and at the top of your resume.

## Editing later
Open any file in a text editor, change the text between the tags, save, and re-upload to GitHub (or edit directly in GitHub's web editor). Colors and fonts are defined once at the top of each file in the `:root` block.

## To update
- Add a case study: copy `case-reliability.html`, change the content, and add a matching card in the "Case studies" section of `index.html`.
- Swap contact details: search each file for `sravani.guntupalli09@gmail.com` and the LinkedIn URL.
